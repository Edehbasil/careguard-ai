# CareGuard AI

CareGuard AI is an AI-assisted compliance and risk monitoring platform designed to help healthcare providers manage regulatory requirements more effectively.

## Problem
Healthcare organisations often rely on manual processes to manage compliance, increasing the risk of missed actions and poor inspection outcomes.

## Solution
CareGuard AI combines structured compliance checklists with AI-assisted document analysis to help providers identify risks early and stay inspection-ready.

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React
- Database: PostgreSQL
- AI: API-based language models
- Cloud: AWS/Azure
